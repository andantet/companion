# Chat

## Configuration

<details>

<summary>Mentions</summary>

Enables chat mentions.

**Type:** boolean\
**Default:** false

</details>

<details>

<summary>Mentions Color</summary>

Changes the color of the mention highlight.\
Only available in the config JSON.

**Type:** integer\
**Default:** `#E7FF54` (`15204180`)

</details>
